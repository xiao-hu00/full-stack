## admin-app

后台管理系统

主要技术栈
- `shadcn/ui-ui` headless UI 框架
- `zustand` 状态管理
- `@tanstack/react-query` 数据获取
- `axios` 数据请求
- `tailwind css` css 框架
- `react-router` 路由管理
- `react-hook-form + zod` 表单管理和校验
